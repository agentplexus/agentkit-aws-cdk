## [v0.2.0] - 2026-03-01

- Module migrated from `agentplexus` to `plexusone` organization
- New CLI tools for one-command AWS AgentCore deployment
- Full AWS Bedrock AgentCore resource creation with Runtime, Endpoint, and Gateway support

### Breaking

- Module path changed from `github.com/agentplexus/agentkit-aws-cdk` to `github.com/plexusone/agentkit-aws-cdk` ([`03aeda8`](https://github.com/plexusone/agentkit-aws-cdk/commit/03aeda8))

## [v0.1.0] - 2025-12-31

### Highlights

- Initial release of AWS CDK constructs for deploying agentkit-based agents to AWS Bedrock AgentCore

### Added

- `AgentCoreStack` CDK construct for deploying multi-agent systems
- `AgentBuilder` fluent API for configuring individual agents with memory, timeout, environment variables
- `StackBuilder` fluent API for composing agents with VPC, observability, and IAM settings
- `CfnIncludeBuilder` for importing existing CloudFormation templates into CDK
- `NewStackFromFile` and `MustNewStackFromFile` for loading configuration from JSON/YAML
- `NewStackFromJSON` and `NewStackFromYAML` for loading configuration from byte data
- Re-exported configuration types from `agentkit/platforms/agentcore/iac`
- VPC creation with optional VPC endpoints for AWS services
- Security group creation for agent communication
- IAM execution role with Bedrock, CloudWatch Logs, ECR, and Secrets Manager access
- Observability configuration support for Opik, Langfuse, Phoenix, and CloudWatch
- Four deployment approaches: CDK Go, CDK+JSON, CfnInclude, Pure CloudFormation

### Documentation

- README with comprehensive usage examples for all deployment approaches
- Example projects in `examples/` directory for each deployment approach
- Configuration reference for StackConfig, AgentConfig, VPCConfig, ObservabilityConfig
